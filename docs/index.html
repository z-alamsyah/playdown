Built for people who live in markdown all day — skills, agents, PRDs, manifests.

🪟

Split panes

VSCode-style splits — drag a tab to any edge, resize, nest. Edit left, preview right.

🔎

Quick open

Fuzzy file finder on ⌘P. Jump anywhere instantly.

🧭

Live outline

Heading navigation opposite the sidebar. Click to jump.

🧩

Frontmatter-aware

YAML frontmatter folded in the editor, rendered as a card in preview.

🔡

JSON tools

Syntax highlighting + one-key format. Highlighted JSON preview.

🖼️

Image preview

PNG, JPG, SVG, GIF & more — rendered inline, no bloat.

🖥️

Built-in terminal

Real multi-session PTY terminal (VSCode-style), Ctrl+`. Lazy-loaded — zero idle cost.

⌨️

playdown CLI

Open any folder from your shell: playdown . — plus --version / --help.

🧜

Mermaid + GFM

Diagrams, tables, task lists, code highlighting in preview.

⚙️

Make it yours

Themes, sidebar side, zoom, rebindable shortcuts, colored title bar.

🪶

Genuinely light

Tauri + system WebView. No bundled Chromium. ~5 MB, ~70 MB RAM.

🆓

No sign-in, free forever

No account, no telemetry, no subscription. Open source (MIT) — just download and go.

Built day-by-day with Claude Code — from "this feels overkill" to a shipping app.

Started as a gripe: full IDEs eating hundreds of MB just to read a .md file. A few iterations later: editor groups, drag-to-split, outline, JSON, images, a configurable orange title bar — and an installer. Open source, MIT.
